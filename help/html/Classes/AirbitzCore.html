  • errorString: error message

Return Value

Discussion

requestOTPReset Launches an OTP reset timer on the server, which will disable the OTP authentication requirement when it expires.

This only works after the caller has successfully authenticated with the server, such as through a password login, but has failed to fully log in due to a missing OTP key.
